# cocos creator 框架（还没想好起什么名字，kxCreator？）

## 核心框架

* 资源管理
  * ResLoader：资源的加载和卸载，支持资源依赖的识别
  * PrefabPool：Prefab的简单池子，用于优化大量重复创建的对象
* 网络框架
* UI框架
* 通用
  * EventManager: 事件机制的支持
  * TaskQueue: 任务队列的支持（确保有序地执行逻辑)
